4 tabs (Today / Recipes / Plan / You) · canonical Today · north-star block · Free + Pro pricing · single Log sheet · streak-as-pip · trust posture on every macro row.

What you're seeing

Three contexts in one row, per surface. Native iOS app (812pt iPhone with notch + tab bar), mobile web (browser chrome at top), desktop web (1280px with 248pt sidebar).

Strategic decisions visible at a glance:

  • 4 tabs: Today / Recipes / Plan / You. No Discover, no More — Discover lives inside Recipes; More merges into You.
  • Canonical Today: day strip → calorie ring → north-star "what to eat next" block → macros remaining → meals → persistent Log FAB. No 3-variant picker, no streak ribbon.
  • North-star block is the second thing the eye lands on after the ring — gradient background, "fits your remaining macros" chip, swipe-to-skip.
  • Streak is a pip next to the date, not a ribbon.
  • Caffeine + alcohol are not on Today (Settings opt-in only).
  • Single Log sheet with 6 sub-tabs (search/barcode/recent/saved/voice/photo). Replaces 8+ entry points.
  • Recipes tab defaults to Library sub-tab, with Discover as a peer.
  • You tab merges Progress + Settings + More. Profile card at top, story-led Progress as headline, settings rows below.
  • Pricing is Free + Pro only. Base is gone.
  • Trust posture: every meal/recipe/macro row carries a coloured source dot (USDA = green, OFF = blue, manual = grey).
  • Onboarding ends with recipe-pick that seeds Library + first plan.
  • Mobile web shows the install hint at the bottom — same canonical Today as native.
  • Desktop web uses the 248pt sidebar with a free-tier upgrade card pinned bottom (matches prototype).
