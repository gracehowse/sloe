What changed (target vs shipped) — same architecture, elevated treatment

  • Ring: gradient stroke (success → success-light) instead of flat colour. Drop-shadow on arc gives it physical depth. Track at 50% opacity (subtle, not aggressive grey). Ring grew 132 → 152pt — air around the number.
  • Centre number: 28pt → 48pt with -0.04em tracking. The "1,247" is the moment — make it the moment. Animated fade-up on first paint.
  • Macros column: type weight ladder elevated. Name in proper weight (was 9pt overline, now 12pt 600 sentence-case). Bar from 4pt to 4pt with proper rounded ends and motion fill on first paint.
  • Hero card: radius 16 → 28, padding 14 → 28×24, depth via subtle radial gradients (primary top-right, success bottom-left at 6-8% opacity). 1px hairline border replaces the 1px solid. Box shadow is real depth, not none.
  • North-star block: same gradient base + tighter geometry. Image gets soft drop-shadow + subtle highlight (looks photographic, not flat). CTA button is dark, not primary — primary energy lives on the ring; CTA is the action handle. Spring physics on hover/press. Pill copy "Cook →" with proper arrow glyph.
  • Section header: gains a "View all" affordance + tighter tracking. "TODAY'S MEALS" not "Today's meals".
  • Meal cards: 12pt → 18pt radius, larger thumbs (40 → 48), photographic gradients with light spot (looks like food, not blocks of colour), timestamps in caption. Hover lifts +1px with shadow change.
  • Source dot: shrinks to 5pt (more refined, less prominent). Source name + meal type + time together: "OFF · Breakfast · 7:42".
  • Streak pip: warmer amber tinted background, hairline border instead of just outline, proper Flame glyph from lucide.
  • Date row: eyebrow "TUESDAY" + display number "28 April" (matches Apple Calendar / Things). Drops the comma format.
  • Tab bar: backdrop blur 20px + 72% white overlay (iOS native). Active tab uses a filled glyph for emphasis.
  • FAB: gradient (primary → primary-light) + larger soft shadow with primary tint glow + spring on hover/press.
  • Sticky condensed header: appears on scroll past the hero. Mini ring + total-remaining number. Backdrop blur. Same iOS-native pattern as Apple Fitness.
  • Typography ladder: every text element uses tabular-nums + cv11 for crisp numerals, proper letter-spacing per size, proper line-heights. No ad-hoc font-size literals.
  • Motion: ring fills on first paint (1400ms spring with bounce). Macro bars fill staggered after the ring (1200ms ease-spring, 600ms delay). Cards lift on hover. CTA spring-scales on press. Sticky header crossfades. Reduce-motion fallback collapses all to instant value swaps.
  • Dark mode: real dark-mode polish — the same tokens but at OLED-friendly black, hairline highlights instead of shadows on cards, primary glow on FAB stays.

What's the same (architecture is locked)

  • 4 tabs, canonical Today, calorie ring + 4-macro readout, north-star block placement, persistent FAB, source-dot trust system, streak as pip, no caffeine/alcohol on Today, descriptive gluten chip language. Architecture stays. Treatment elevates.
