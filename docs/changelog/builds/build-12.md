# Build 12 (1.0.0 #12) — 2026-05-01

## New

- **Today — micronutrient widget (audit gap #1).** Below the macro
  tiles on Today, four scrollable tiles surface fibre, iron, vitamin D
  and sodium against the FDA 2020 Daily Value reference set. Sodium
  ramps from green → amber at 80% to red at 100% (it's a limit, not a
  target). Closes the "Suppr is just a macro tracker" feedback loop
  for users coming from Cronometer / MacroFactor without expanding
  the surface — the full all-nutrients view still ships via the
  Nutrients link in the macro tile header.
  Mobile + web (`apps/mobile/components/today/TodayMicrosWidget.tsx`,
  `src/app/components/suppr/today-micros-widget.tsx`).
