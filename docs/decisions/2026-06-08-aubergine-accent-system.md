# Sloe accent = single aubergine-violet `#5B3B6E`, rationed Julienne-style

**Date:** 2026-06-08
**Status:** Resolved
**Area:** Brand / design system
**Decider:** Grace (research routed to `design-director` + `brand-manager`; treatments approved from rendered boards)
**Supersedes:** `docs/decisions/2026-06-08-functional-accent-clay-not-damson.md` (clay — same day, now reversed)

## Decision

Sloe runs **one accent: aubergine-violet `#5B3B6E`**, used **sparingly** on a white / warm-grey neutral canvas, with food photography as the colour hero — the Julienne editorial model. **Clay is dropped from the chrome** and survives only as the **carbs macro colour**.

### The scale (re-chroma'd from the flat damson)
| Token | Hex | Role |
|---|---|---|
| Plum Ink | `#3B2A4D` | wordmark, serif headings, deepest depth |
| **Plum (accent base)** | **`#5B3B6E`** | FAB fill, filled conversion CTAs, selected-pill tint base |
| Plum Lift | `#7E5C92` | gradient top, pressed/glow, dark-mode/active lift |
| Plum Solid | `#4E3260` | text / icon / link / **outline border** on light (≈8.7:1 AA) |
| Plum Soft | `rgba(91,59,110,0.10)` | selected pills, segmented active, nudge tint |

The previous damson `#6A4B7A` measured **7% saturation** (a grey-violet — quantified by `design-director`); `#5B3B6E` is a real, deliberate aubergine.

### Component treatment system (approved from `docs/prototypes/sloe-component-treatments.html`)
Colour drops as emphasis drops — ration the accent like Julienne:
- **FAB** → filled aubergine (the ONE filled moment).
- **Primary inline CTA** (Cook this, Log, Generate, Start fast, Save) → **aubergine OUTLINE** (1.5px, accent as a line, not a slab).
- **Conversion CTA** (paywall, onboarding Continue) → filled aubergine (the exception).
- **Secondary** → off-white `#F6F5F2` fill, ink label.
- **Tertiary / dismiss** → text-only.
- **Destructive** → red. **OAuth** → ink. (Both unchanged.)
- **Filter pill** selected → aubergine soft-tint / unselected → off-white. **Segmented** active → lift + aubergine label (the **pricing billing toggle** uses the white-lift form; the **onboarding metric/imperial unit-toggle** uses the aubergine **soft-tint** form so the web `Segmented` and mobile `MobileSegmented` stay in lockstep — never a solid accent fill). **Plan-selector** active row → aubergine soft-tint + 2px accent ring (radio dot + "BEST VALUE" badge keep the solid accent). **Pro** → aubergine tint.
- **Fit chip / "On track"** → **sage, unchanged** (success semantic, never the accent).
- **Cards** → warm-grey / white; photography carries the colour.

## Why (research-grounded)

1. **Clay was undifferentiated.** Clay `#C8794E` sits in the crowded warm-orange recipe lane Julienne / Lose It! / Paprika already own (`docs/brand/2026-06-07-secondary-colour-exploration.md`). Aubergine is unowned in the category — and **a sloe is a deep purple-red berry**, so the name hands you the colour.
2. **The editorial-app pattern** (Julienne, NYT Cooking, Mela, Things, Aesop, Kinfolk/Cereal): neutral canvas + serif/photography carrying hierarchy + **ONE accent used by rarity**. Sloe was breaking this with two colours (clay + plum) used broadly.
3. **The "flat purple" was chroma starvation**, not the hue — damson at 7% saturation reads grey, plum `#3B2A4D` reads near-black. Fix = a richer shade + tonal scale + depth + restraint, not "less purple."
4. **Appetite/warmth** is carried by hyperreal food photography + warm-grey cards + clay-as-carbs — not by the button colour (Julienne proves a recipe app doesn't need an orange accent to feel appetising).
