Five principles

What we hold to when in doubt.

Restated up front so every later decision can be checked against them. If a future change violates one of these without a written exception, it doesn't ship.

No. 01

Restraint over flourish.

Every visible element earns its place. ALL-CAPS overlines, gradient placeholders, decorative icons, persistent celebratory toasts — gone unless they carry information no other element does.

No. 02

Weight does the hierarchy.

Family changes (serif vs sans) are reserved for ~3 specific editorial moments per surface. Within a card, ONE family. Inter at varied weights does 95% of the work.

No. 03

One accent. Six places. No exceptions.

Terracotta appears on: streak count · ring progress · FAB · "Log meal" CTA · today marker · active tab-bar item. Everywhere else uses greyscale or weight contrast.

No. 04

Interpretive copy beats raw numbers.

Gentler Streak's Apple-commended move. "REMAINING 380" → "380 kcal left for dinner." The calorie ring centre, macro captions, Progress headlines all switch to sentence-form interpretation.

No. 05

Cook Mode is where we win an Apple Design Award.

Visual polish makes screenshots better; Cook Mode excellence makes the app sticky. Voice listener un-deferred, wink-nav added, Live Activity timers, iPad two-panel layout.

Before · after — the Today screen

What changes, side by side.

Before · current
  • Persistent green "You're all set!" toast on every screen
  • Dev N 2 Issues badge in TestFlight captures
  • Date written twice — overline JUN 16 · TUESDAY + h1 Tue 16 Jun
  • Page bg #f4f5f7 cool grey; card bg #ffffff
  • Blue #4c6ce0 in 8-12 places per screen as wallpaper
  • ALL-CAPS overlines on every section (HOUSEHOLD, MEAL, LAST 30 DAYS)
  • Macro tiles with decorative icons (steak / wheat / blood-drop / leaf)
  • "REMAINING 380" raw-number centre + "98 g remaining" × 4 captions
  • Empty-state cards taking 25%+ of the screen
  • Sub-tab pills + filter pills + sort chips all looking identical
After · v1
  • Toast auto-dismisses on first scroll; never persists past welcome session
  • Dev chrome locked out — release builds verified clean
  • H1 "Today" carries all the date semantics. No overline duplication.
  • Cream #f6f3ee page bg / warm-black #0e0d0b dark
  • Terracotta #a3522a in EXACTLY six places. Everywhere else greyscale.
  • Overlines kept only on system-nav (TODAY day-strip eyebrow)
  • Macro tiles label-only (Gentler Streak discipline — trust the type)
  • Ring centre: "556 kcal left for dinner" — sentence interpretation
  • Empty states are sentences. One line + one action.
  • Visual hierarchy: tab = sub-tab → filter pill → sort chip via weight + size, not identical pills

Decisions Grace owns

Eight calls needed before V1 starts.

D1

Approve the synthesised direction.

The five principles + token system + phased plan + reference benchmarks above. Sign-off here means V0.6 ships and V1 starts.

D2

Approve the £1,500 photography commission.

Three-tier strategy — 25 commissioned hero shots + 75 licensed + AI long-tail. Brief due before V9 starts. Brand-warmth pillar #1.

D3

Approve the £500–1k illustration commission.

Custom character or geometric shape (Yorhart-equivalent). Style call: abstract bowl-ish shape or character-with-personality? Brief due before V10.

D4

Approve the new calorie-ring colour mapping.

Empty = faint hairline accent (drops indigo→pink gradient). Under = terracotta (drops success-green). Over = destructive red (kept). Overrides current memory; needs memory update too.

D5

Approve the persistent toast kill rule.

Auto-dismiss on first scroll OR never render past welcome session. Either works; pick the rule.

D6

Confirm: no greeting, no masthead, no signature.

"Good evening, Grace" / "No. 142" / "tare · vol. i · spring" all stay dropped per chat round 7. Locking explicitly.

D7

Approve iOS 26 Liquid Glass adoption.

Gentler Streak shipped this on iOS 26. Free polish, requires iOS 26 + Xcode 16+ targets.

D8

Approve interaction-track ordering.

Un-defer Cook Mode voice listener (I1) ahead of TrueDepth wink-nav (I2) — voice has wider audience; TrueDepth needs iPhone X+.
