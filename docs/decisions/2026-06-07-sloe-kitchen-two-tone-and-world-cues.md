# Decision: Sloe kitchen two-tone variant, world cues, and interior reference

- **Date:** 2026-06-07
- **Area:** Brand / Imagery
- **Status:** Resolved
- **Source:** @jadewilson.f kitchen/pantry/dining reels, surfaced by Grace (two consecutive drops, same session)
- **Decision-maker:** Grace (approved), formalised by `brand-manager`

---

## Context

Grace shared five frames from @jadewilson.f (a cream island kitchen with rush stools, a butler's pantry,
a dining room with a black linear chandelier). The world reference board
(`docs/brand/sloe-world-reference-board.md`, Entry 1 + 1b) assessed them as **convergence with the locked
Sloe test kitchen spec, not drift** — every frame confirmed the existing cream/brass/marble/daylight DNA.
The reels surfaced seven refinements that the spec did not yet encode. Grace approved all seven.

---

## Decisions (7)

### 1. Unlacquered/aged brass as the explicit hardware spec

**What:** all brass hardware in the locked test kitchen (cup-pulls, knobs, faucet, pot-filler, wall-rail)
is specified as **unlacquered/aged warm brass** with a soft, lived-in sheen and slight natural patina.
Never bright polished mirror-gold, never chrome.

**Why:** the reference reels show warm, slightly patinated brass throughout (the long fridge bar-pull
especially). The prior spec said "brass" without finish guidance, risking generated kitchens that render
cold mirror-gold. The aged finish maps to the brand's warm Honey/Clay accent register.

**Files:** house-style §3 (intro paragraph + hardware spec callout), prompt block §8.1.

### 2. One restrained matte-black architectural accent permitted

**What:** a single matte-black architectural accent (lighting fixture, window mullion, thin shelf bracket)
is permitted in Tier-1 kitchen and lifestyle frames as a graphic anchor. Never a black surface, cabinet,
countertop, or backdrop. Maps to Ink `#221B26`.

**Why:** the dining-room frames show a matte-black linear chandelier and black window mullions as the
single dark note against all-cream. It reads considered, not loud — the home-decor analogue of using Ink
as a sparing graphic anchor on the Oat ground.

**Files:** house-style §3 (matte-black accent rule), reviewer checklist §9.

### 3. Cream-perimeter + dark-oak-island two-tone approved as a sanctioned kitchen variant

**What:** the locked test kitchen now has an approved **variant** where the island is dark stained oak
(warm, deep mid-tone wood) with a white quartz top and aged brass hardware, while the perimeter stays
cream Shaker. This is a second option for marketing-frame variety, not a replacement for the all-cream
room.

**Why:** the reference reels show this two-tone layout as a deliberate mid-tone contrast anchor. The dark
wood maps tonally toward the brand's warm-wood accents (near Ink in weight) without being as dark as a
black surface. It gives the social feed visual range while keeping the recognisable DNA.

**Files:** house-style §3.1b (new subsection), prompt-template §3c (two-tone variant note).

### 4. Architecture cues added to the Tier-1 locked-kitchen description

**What:** three architectural details are now part of the locked test kitchen spec:
- Plaster arched range niche + plaster range hood (soft, hand-troweled, no hard edges)
- White apron-front / fireclay farmhouse sink
- Copper pots on a brass wall-rail beside the range (rail = aged brass per decision 1)

**Why:** these appear repeatedly in the reference reels and add the warm-European, slightly-editorial
architectural character that distinguishes the Sloe kitchen from a generic white-marble set. They are
convergent with the existing spec (copper pots were already specified; the plaster and sink sharpen the
architectural language).

**Files:** house-style §3 (architecture cues callout).

### 5. Lifestyle prop kit expanded for Template D (Tier 1 only)

**What:** the following props are added to the sanctioned Template D lifestyle prop kit:
- Rush/rattan counter stools with linen cushions + tie-bows
- White linen empire/cone pendant shades on brass chains
- Gold-framed pastoral landscape oils or still-life floral oils (thin frames, moody/warm tones)
- Ceramic urns with fresh stems (tulips, hydrangea)
- A worn wooden dough bowl
- Glass storage canisters
- Beadboard/shiplap ceiling + classic millwork (background architecture detail)

**Why:** these enrich the lifestyle set (app resting in a warm kitchen) without touching the per-dish
plating prop list (which stays "2-4 functional props, garnish = the dish's own ingredients"). They are
Tier-1/marketing only and give Template D frames the warmth and character of a real, lived-in kitchen.

**Files:** prompt-template §3b (Template D, lifestyle prop kit section).

### 6. @jadewilson.f named as the pinned interior/world reference

**What:** @jadewilson.f is the named interior/world reference for the Sloe test kitchen set, architecture
cues, hardware finish, and Template D lifestyle frames — parallel to @_foodstories_ /
@thelittleplantation for food photography.

**Why:** two consecutive same-creator reference drops from Grace (both in the same session) confirmed
strong convergence with the locked spec. Pinning a named reference gives the "world" a fixed touchstone
the way the food handles give the dishes one.

**Files:** house-style §10b (new named-reference-handles subsection), prompt-template §3b (Template D
class description).

### 7. Voice-swipe lines logged for marketing use

**What:** five caption/voice lines from @jadewilson.f's slow-living content are logged as marketing voice
swipe entries:
- "things in my home I'll never regret investing in"
- "a slow Sunday morning"
- "the older I get the more I romanticize staying home"
- "making my home feel softer one little detail at a time"
- "created the home I kept saving on Pinterest"

**Route:** social, onboarding, landing copy only. Never in-app chrome.

**Why:** the throughline (home as unhurried sanctuary, softness, permanence, slowness) is the Sloe
positioning rendered in a different category. These lines mine for landing/onboarding/social tone without
ever being used verbatim (they are swipe, not copy).

**Files:** sloe-brand.md §5 (new voice-swipe subsection).

---

## Rationale (cross-cutting)

All seven decisions are **convergence with the locked spec**, not drift. The reference reels showed the
Sloe test kitchen *as it already existed on paper* — rendered in a real home. The decisions sharpen,
enrich, and add variety to the spec without changing the DNA (cream/brass/marble/daylight/restraint). The
two-tone variant (decision 3) is the most structurally significant change, and it is deliberately framed
as an approved variant, not a replacement, to preserve the recognisable all-cream room as the default.
