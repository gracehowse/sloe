# Production Readiness Audit — Findings

**Date:** 2026-07-05
**Stage:** production_readiness
**Total findings:** 32

## Single highest-risk gap

**The two channels that are supposed to tell Grace when something is broken are both dark: mobile crash reporting (Sentry) has recorded zero events in 90 days because no build ever ships a DSN, and web product analytics (PostHog) has been silently uninitialised in production because its env var was never set — so the primary iOS surface can crash and the web surface can misfire with nobody finding out except via a user complaint.**

## 10k-users-tomorrow verdict: FALLS OVER

If 10,000 users landed tomorrow, the product does not gracefully degrade — it hits hard, undetected stops. **Where it falls over:** Supabase is on the Free plan, which does not throttle at its limits, it hard-stops (500MB DB → read-only, breaking signups and food logging product-wide; ~5GB/month egress plausibly exceeded in a single day of recipe/photo image traffic). Vercel is on Hobby, which is licensed non-commercial and can simply pause the deployment rather than absorb overage. Neither vendor has a usage alarm configured, and Vercel's error-rate alarm cannot even be configured on Hobby — so the first signal of a total outage would be users telling Grace, not a dashboard. Layered on top: mobile has zero crash telemetry (PRA-001/IM-03), the one wired security cron has been silently failing since 2026-07-02 (PRA-002/IM-05), web PostHog and web Stripe are both dead in production (IM-01, IM-02), and the DR restore path has never been rehearsed against a database that has never needed to survive real load (PRA-003/IM-07). The AI and food-vendor cost/quota guardrails are genuinely well-built and would degrade gracefully rather than break — but they're sized for 1,000 DAU, not 10,000, so they trip within hours and narrow nutrition-match quality or 503 AI logging exactly when first impressions matter most (PRA-010/011, IM-10/11). None of this requires a hypothetical worst case: it is the documented behaviour of the plans currently in production.
