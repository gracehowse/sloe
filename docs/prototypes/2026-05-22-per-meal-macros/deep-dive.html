Per-meal macros — tap-to-deep-dive sheet proposals

Goal: when user taps the slot (or a meal row), open a sheet showing kcal + every macro + per-food contribution. Three approaches below — ranked from "summary-first" to "contribution-first".

D1 — Pie + foods list
Macro pie up top (kcal split). Below: each food as a row with its own per-macro pips. "How did this meal break down" answered at-a-glance, with per-food drill if needed.
