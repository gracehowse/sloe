My recommendation

Ship Option A (#fbfaf7, web parity) as the default. The real defect here isn’t the colour — it’s that mobile drifted to opaque while web stayed at 40%. Option A fixes that divergence in one line, keeps the hero’s “one soft-filled region” intent from ENG-740, and immediately reads more premium on your phone.

If after seeing it in the sim you want to push further, Option C (hairline, no fill) is the more daring, most-premium move and is the most faithful to our own “warmth lives in ink + hairlines, not fills” lock — but it slightly weakens the hero as a distinct zone, so I’d only go there if A still feels too fill-y.
