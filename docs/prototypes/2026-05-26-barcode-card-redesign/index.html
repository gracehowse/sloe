What changed

Hero calories. 265 kcal is the single largest number (27px/800, macro-green) — the value you’re logging. Food name, macros and amount no longer compete.

4-tile macro band replaces the dense “265 kcal P:53g C:15g F:0g” line. Same as the Log-sheet scanner; web dialog adopts it too (it’s currently a text line).

One portion control. The “By grams / By serving” segmented toggle is gone — unit is a tap-popover on the amount (PortionPicker). One way to set the number, not three.

Quick chips are sorted, deduped, consistently formatted (from buildPickerOptions) — no more 170g · 100 g · 50 g filler chaos.

One primary CTA. Green Log · 500 g; re-scan demoted to a 44² icon.

Dignified correction. A pencil top-right + a friendly “Numbers look off? Fix them ›” — replaces the grey underlined “This is wrong — edit and update.” (final wording pending copy-review.)

Trust banner inline (right) — the plausibility/over-scale warning is an in-card amber banner, not a blocking system alert. Soft-flag, never blocks.

Solid surface. No more dark-glass card floating on the live camera — camera collapses once a result lands.

Open calls for you
