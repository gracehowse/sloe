Suppr 8-colour palette — Apple Photos anchor

Constrained palette: every colour anywhere in the app (macros, slots, accents, semantic states, marketing) maps to one of these 8 slots. Same tonal family — saturated but not eye-bleeding, all sitting at roughly the same brightness. Grace's brief 2026-05-22: "they are all the right shade / tone … provides a comfortable limit so we don't go creating too many colours and is a good palette tonally."

Eight slots, every colour in the app fits.

Going clockwise from 12 o'clock: magenta · orange · yellow · lime · green · cyan · blue · purple. Each ~45° apart on the wheel; same saturation + brightness family. Any new colour need that comes up in design (a new tab tint, an experimental state) must map to one of these eight — or it doesn't ship.

7 of the 8 are already in the code. Lime + purple are net-new and need a home; suggestions below.
