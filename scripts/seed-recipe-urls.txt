# How this list is consumed:
#   - scripts/seed-discover-recipes.ts        → imports each URL into the
#     `recipes` + `recipe_ingredients` tables under the seeded

#     URLs by `source_url`).
#   - scripts/delete-seeded-recipes.ts        → if you need to nuke the
#     batch, this script already deletes any recipe whose `source_url`
#     matches a line in this file (see docs/operations/scripts.md).
#
# Lines starting with `#` and blank lines are ignored.
